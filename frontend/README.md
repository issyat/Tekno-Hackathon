# Frontend (Vite + React)

A minimal React 18 + Vite setup with fast refresh, created in `frontend`.

## Quickstart (Windows PowerShell)

```powershell
cd "c:\Users\ismai\OneDrive\Bureau\Tekno-Hackathon\frontend"
npm install
npm run dev
```

Open the URL printed by Vite (typically http://localhost:5173).

## Scripts
- `npm run dev` – start dev server
- `npm run build` – build for production
- `npm run preview` – preview the production build

## Notes
- Requires Node.js 18+ for Vite 5.
- Uses `@vitejs/plugin-react` for Fast Refresh and better JSX DX.
